Symbol Reference (ISO 5807 Flowchart Standard)

Rounded Rectangle / Oval (Terminal): Marks Start, End, and Commit ✓ sub-process completions. Blue pill = Admin path; Green pill = Cashier path.
Rectangle (Process): Any action step — white = shared process, blue fill = admin feature, green fill = cashier feature, orange border = backend POST/AJAX request, purple = sync event.
Diamond (Decision): Branching point with Yes/No outcomes — Credentials Valid?, Check Role, Resellable Item?, Same Browser Session?
Wavy-bottom Rectangle (Document): Printable report output — Sales History Report, Cashier Report (per-shift), Shift Summary Report.
Dashed Arrows → Sync: A completed transaction triggers the real-time sync layer (not a sequential step; an event broadcast).
